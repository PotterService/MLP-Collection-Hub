# MLP Collection Hub

A central hub for Bow's fan-made Kayou My Little Pony collector tools.

This website acts as a directory and landing page for MLP collector projects, making it easy to find live websites, GitHub repositories, and standalone set trackers.

---

## Live Website

https://mlp.potterservice.com

---

## Included Projects

### BP01 Fantasy Wonderland Collector

Standalone fan-made collector tool for the Kayou My Little Pony Card Game BP01 set.

Features:

- Card image gallery
- Owned tracking
- Duplicate tracking
- Wishlist support
- Pull odds guide
- CSV exports
- JSON backups
- Print-friendly binder view

Live Site:

https://bp01.potterservice.com

GitHub:

https://github.com/PotterService/BP01-Fantasy-Wonderland-Collector

---

### MLP Kayou Collection Tracker

A fan-made collection manager for the Kayou My Little Pony Trading Card Game.

Track:

- Collection progress
- Duplicates
- Wishlist cards
- Set completion
- Release years
- Exportable checklists
- Local backups

Live Site:

https://mlptracker.potterservice.com

GitHub:

https://github.com/PotterService/MLP-TCG-Collection-Vault

---

## Purpose

The goal of this hub is to provide a single location where collectors can easily access:

- Collection trackers
- Set-specific tools
- Binder tools
- Checklist projects
- Future collector utilities

As more collector tools are released, they can be added to this hub.
